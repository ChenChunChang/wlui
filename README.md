## [文档地址](http://docs.wlui.com.cn/)

## Install
```shell
npm install wlui -S  

# or

yarn add wlui
```

## Quick Start
``` js
import Vue from 'vue'
import WlUi from 'wlui'

Vue.use(WlUi)

// or

import {
  WlGantt,
  WlExplorer
  // ...
} from 'wlui'

Vue.use(WlGantt)
Vue.use(WlExplorer)
```
